[size=5][color=#4CAF50]‚ùì FAQ[/color][/size]

[b]Q: Will this improve my game performance?[/b]
A: Packing files into BSA/BA2 archives can help with load times and reduce file system overhead, but results vary based on your system, mod setup, and how many loose files you had. This tool automates the packing process correctly - it doesn't guarantee specific performance numbers.

[b]Q: Is this safe to use?[/b]
A: Yes. The tool never modifies your original files. It creates copies and works in temporary directories. Everything is open-source for verification.

[b]Q: Do I need Python?[/b]
A: Not with the bundled release. Otherwise, yes - Python 3.7 or later.

[b]Q: Does this work with MO2?[/b]
A: Yes. Install the packed archives as mods in MO2. Keep loose files in a separate mod with higher priority.

[b]Q: What about script files?[/b]
A: Script files are kept loose by default for compatibility. SKSE/F4SE folders are never packed.

[b]Q: Can I use this for commercial projects?[/b]
A: Check the MIT license, but it's generally permissive. Don't claim to have created this tool.
